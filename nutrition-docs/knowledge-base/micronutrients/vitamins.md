# Vitamins

## Overview

Vitamins are organic compounds essential for metabolic processes. Required in small amounts but critical for health. Divided into fat-soluble (A, D, E, K) and water-soluble (B-complex, C).

---

## Fat-Soluble Vitamins

Stored in body fat and liver. Can accumulate to toxic levels if over-supplemented.

### Vitamin A (Retinol)

| Property | Value |
|----------|-------|
| RDA | 900 mcg (men), 700 mcg (women) |
| Upper Limit | 3,000 mcg |
| Functions | Vision, immune function, skin health, cell growth |
| Deficiency | Night blindness, dry skin, weak immune system |
| Toxicity | Liver damage, birth defects, bone loss |

**Best Sources**:
| Food | Serving | Vitamin A |
|------|---------|-----------|
| Beef liver | 85g | 6,582 mcg |
| Sweet potato | 1 medium | 1,403 mcg |
| Carrots | 1 medium | 509 mcg |
| Spinach (cooked) | 100g | 469 mcg |
| Eggs | 1 large | 80 mcg |

*Note: Plant sources contain beta-carotene (provitamin A), which converts at ~12:1 ratio.*

---

### Vitamin D (Cholecalciferol)

| Property | Value |
|----------|-------|
| RDA | 600-800 IU (15-20 mcg) |
| Optimal Range | 1,000-4,000 IU daily |
| Upper Limit | 4,000 IU (some experts say 10,000 IU) |
| Functions | Calcium absorption, bone health, immune function, mood |
| Deficiency | Weak bones, depression, frequent illness, fatigue |

**Deficiency is extremely common** - especially in northern latitudes, darker skin, indoor lifestyles.

**Best Sources**:
| Source | Amount | Vitamin D |
|--------|--------|-----------|
| Sunlight | 15-30 min | 10,000-20,000 IU |
| Cod liver oil | 1 tsp | 450 IU |
| Salmon | 100g | 570 IU |
| Fortified milk | 240ml | 120 IU |
| Eggs | 1 large | 44 IU |
| Supplement | 1 softgel | 1,000-5,000 IU |

**Testing**: Blood test for 25(OH)D. Optimal range: 40-60 ng/mL.

### Vitamin K

| Property | Value |
|----------|-------|
| RDA | 120 mcg (men), 90 mcg (women) |
| Functions | Blood clotting, bone metabolism, calcium regulation |
| Deficiency | Excessive bleeding, weak bones |

**Two forms**:
- **K1 (Phylloquinone)**: Leafy greens - blood clotting
- **K2 (Menaquinone)**: Fermented foods, animal products - bone and heart health

**Best Sources**:
| Food | Serving | Vitamin K |
|------|---------|-----------|
| Kale | 100g | 817 mcg (K1) |
| Spinach | 100g | 483 mcg (K1) |
| Natto | 100g | 1,000 mcg (K2) |
| Hard cheese | 100g | 76 mcg (K2) |
| Egg yolks | 1 large | 32 mcg (K2) |

## Water-Soluble Vitamins

Not stored in body; excess excreted in urine. Need regular intake.

### Vitamin C (Ascorbic Acid)

| Property | Value |
|----------|-------|
| RDA | 90 mg (men), 75 mg (women) |
| Optimal Range | 200-500 mg |
| Upper Limit | 2,000 mg |
| Functions | Antioxidant, collagen synthesis, immune function, iron absorption |
| Deficiency | Scurvy, slow healing, weak immune system |

**Best Sources**:

| Orange | 1 medium | 70 mg |
| Kiwi | 1 medium | 64 mg |
| Broccoli (cooked) | 100g | 65 mg |
| Strawberries | 100g | 59 mg |

### B12 Deep Dive

Critical vitamin - deficiency causes:
- Fatigue and weakness
- Nerve damage (tingling, numbness)
- Cognitive issues
- Anemia

**At-risk groups**: Vegans, elderly, those with gut issues.

## Vitamin Interactions

| Pair | Interaction |
|------|-------------|
| Vitamin D + K2 | D increases calcium absorption; K2 directs it to bones |
| Vitamin C + Iron | C enhances iron absorption |
| Vitamin D + Magnesium | Magnesium needed to activate D |
| Vitamin E + C | C regenerates E |
| B vitamins | Work synergistically; take as complex |

## Supplement Guidelines

| Vitamin | Supplement? | Notes |
|---------|-------------|-------|
| A | Usually no | Easy to get from food; risk of toxicity |
| D | Often yes | Most people deficient; test levels |
| E | Usually no | Food sources sufficient |
| K | Sometimes | K2 if not eating fermented foods |
| C | Optional | Food sources usually adequate |
| B-complex | Situational | Useful for vegans, elderly, high stress |
| B12 | For vegans | Essential if no animal products |

## Signs of Deficiency

| Symptom | Possible Deficiency |
|---------|-------------------|
| Fatigue | B12, D, Iron |
| Muscle weakness | D, B12, E |
| Poor wound healing | C, Zinc |
| Night blindness | A |
| Bleeding gums | C |
| Cracks at mouth corners | B2, B6 |
| Hair loss | Biotin, D, Zinc |
| Depression | D, B6, B12, Folate |
